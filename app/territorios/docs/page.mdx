# Documentación de Territorios API

Bienvenidos a la API para obtener la codificación territorial de Costa Rica. Este documento proporciona información detallada sobre nuestra API,
incluyendo una descripción general de sus características, instrucciones de uso y ejemplos de solicitudes.

## ¿Qué es la Codificación Territorial?

La codificación territorial se refiere a un sistema de clasificación y organización de las divisiones políticas y geográficas de un país.
En el caso de Costa Rica, esta información es esencial para diversas aplicaciones, como la gestión gubernamental o el envío de paquetes.

## Características Principales de la API

Nuestra API ofrece una manera fácil y rápida de acceder a la codificación territorial de Costa Rica. Está diseñada para proporcionar
información precisa y actualizada sobre las divisiones políticas del país.

1. **Consulta provincias**

2. **Consulta cantones**

3. **Consulta distritos**

4. **Consulta barrios** (en desarrollo)

5. **Consulta códigos postales**

## ¿Cómo empezar?

Para comenzar a utilizar nuestra API, dirígete a nuestro [Quickstart](https://territorios.fetchcr.com/docs/quickstart)
Por el momento nuestros endpoints están abiertos al público, pero en el futuro se requerirá una clave de acceso para utilizarlos y graduar el
uso de la API.
