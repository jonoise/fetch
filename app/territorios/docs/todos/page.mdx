# Todos los recursos

Por medio de este endpoint se pueden obtener todos los recursos de la API en un sólo request. Se enviarán los recursos
tal y como los tenemos en nuestro sistema, por lo que será tu responsabilidad mapearlos, filtrarlos y ordenarlos.

## Cómo se conforman los recursos

Los recursos que maneja la api son los siguientes:

- **Provincias**: Una **lista** de provincias.

- **Cantones**: Un **objeto** con la identificación territorial de la **provincia** como clave y una lista de cantones como valor.

- **Distritos**: Un **objeto** con la identificación territorial de la **provincia** y del **cantón** como clave y una lista de distritos como valor.
